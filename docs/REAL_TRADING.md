Real Trading Implementation Notes

## ⚠️ ВАЖНО: Реальная торговля активирована!

Бот теперь может выполнять реальные swap транзакции на DeDust и STON.fi.

### Как включить реальную торговлю:

В файле `.env` установите:
```
ENABLE_REAL_TRADING=true
```

По умолчанию стоит `false` (режим симуляции).

### Что реализовано:

#### STON.fi (Рекомендуется)
- ✅ Реальные swap транзакции TON -> Jetton
- ✅ Защита от проскальзывания (5% slippage)
- ✅ Использует Router V2
- ✅ Поддерживаемые токены: USDT, SCALE (настраивается в config.py)

#### DeDust
- ⚠️ Базовая реализация
- ⚠️ Требует дополнительной настройки пулов
- ⚠️ Рекомендуется использовать STON.fi

### Безопасность:

1. **Начните с малых сумм** (0.1-0.5 TON)
2. **Проверьте баланс** перед запуском: `python check_balance.py`
3. **Мониторьте транзакции** в explorer: https://tonscan.org
4. **Держите резерв** минимум 0.5 TON для gas fees
5. **Используйте отдельный кошелек** для экспериментов

### Настройка лимитов:

В `.env`:
```
MIN_TRADE_AMOUNT=0.1    # Минимальная сумма сделки
MAX_TRADE_AMOUNT=1.0    # Максимальная сумма сделки
CHECK_INTERVAL=60       # Интервал между проверками (секунды)
```

### Как работает swap:

1. AI анализирует рынок
2. Принимает решение о сделке
3. Бот строит swap payload
4. Отправляет транзакцию на DEX router
5. DEX выполняет обмен
6. Токены приходят на ваш кошелек

### Проверка транзакций:

После каждой сделки проверяйте:
```bash
python check_balance.py
python cli.py trades
```

Или в TON explorer:
https://tonscan.org/address/ВАШ_АДРЕС

### Поддерживаемые пары:

Настраиваются в `config.py`:
- TON/USDT
- TON/SCALE

Для добавления новых токенов:
1. Найдите contract address токена
2. Добавьте в `config.TOKENS`
3. Перезапустите бота

### Отключение реальной торговли:

В `.env` установите:
```
ENABLE_REAL_TRADING=false
```

Бот вернется в режим симуляции.

### Troubleshooting:

**Транзакция не прошла:**
- Проверьте баланс (достаточно ли TON)
- Проверьте gas fees (минимум 0.1 TON на транзакцию)
- Проверьте ликвидность пула на DEX

**Получили меньше токенов чем ожидали:**
- Это нормально из-за slippage
- Проверьте цену на момент сделки
- Возможно высокая волатильность

**Ошибка "Unknown token":**
- Токен не настроен в config.py
- Добавьте адрес токена в TOKENS

### API Configuration:

Бот использует Kiro.cheap API для Claude:
- Base URL: https://api.kiro.cheap (БЕЗ /v1!)
- Model: claude-sonnet-4-5-20250514
- Max tokens: 2048

Убедитесь что ваш API ключ начинается с `sk-aw-`

**ВАЖНО**: SDK автоматически добавляет `/v1` к base URL, поэтому указывайте только `https://api.kiro.cheap`
