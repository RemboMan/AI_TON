# AI TON Trading Bot - Реальная торговля

## ⚠️ ВАЖНО: Реальная торговля активирована!

Бот выполняет реальные swap транзакции на DeDust.

### Как включить реальную торговлю:

В файле `.env` установите:
```
ENABLE_REAL_TRADING=true
```

По умолчанию стоит `false` (режим симуляции).

### Что реализовано:

#### DeDust ✅
- ✅ Реальные swap транзакции TON → Jetton
- ✅ Реальные swap транзакции Jetton → TON (продажа)
- ✅ Защита от проскальзывания (1% slippage)
- ✅ Автоматический расчет min_out на основе резервов пула
- ✅ Поддерживаемые токены: USDT, STON, DUST, wsTON, GOMINING 

### Безопасность:

1. **Начните с малых сумм** (0.1-0.5 TON)
2. **Проверьте баланс** перед запуском: `python check_balance.py`
3. **Мониторьте транзакции** в explorer: https://tonscan.org
4. **Держите резерв** минимум 0.5 TON для gas fees
5. **Используйте отдельный кошелек** для экспериментов

### Настройка лимитов:

В `.env`:
```
MIN_TRADE_AMOUNT=0.1    # Минимальная сумма сделки
MAX_TRADE_AMOUNT=1.0    # Максимальная сумма сделки
CHECK_INTERVAL=60       # Интервал между проверками (секунды)
```

### Как работает swap:

1. AI анализирует рынок и балансы
2. Принимает решение о покупке или продаже
3. Бот запрашивает резервы пула
4. Рассчитывает min_out с 1% slippage
5. Строит swap payload
6. Отправляет транзакцию на DeDust Native Vault
7. DeDust выполняет обмен
8. Токены приходят на ваш кошелек

### AI может принимать решения:

- **Покупка**: TON → Token (когда цена выгодная)
- **Продажа**: Token → TON (когда есть прибыль)
- **Держание**: Ждать лучших условий

AI видит все ваши балансы и может решить продать токены для фиксации прибыли.

### Добавление новых токенов:

1. Найдите jetton master address токена
2. Добавьте в `config.TOKENS`
3. Найдите pool address на DeDust
4. Добавьте в `dex_handler.py` → `KNOWN_POOLS`
5. Перезапустите бота

### Отключение реальной торговли:

В `.env` установите:
```
ENABLE_REAL_TRADING=false
```

Бот вернется в режим симуляции.

### Troubleshooting:

**Транзакция не прошла:**
- Проверьте баланс (достаточно ли TON)
- Проверьте gas fees (0.3 TON на транзакцию)
- Проверьте ликвидность пула на DeDust

**Получили меньше токенов чем ожидали:**
- Это нормально из-за 1% slippage
- Проверьте цену на момент сделки
- Возможно высокая волатильность

**Ошибка "Pool address not found":**
- Токен не имеет пула на DeDust
- Добавьте pool address в KNOWN_POOLS
- Или используйте другой токен
